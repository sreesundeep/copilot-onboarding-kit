## What is Wyra?

**Wyra** is a **GTM (Go-To-Market) Intelligence Layer** — a pre-CRM platform that converts market signals and ecosystem intelligence into targeted outreach, with humans in control at every decision point.

Think of it as: **Your company's intelligence + AI execution + human judgment = qualified pipeline.**

## Your First Week

| Day | What to Do |
|-----|-----------|
| **Day 1** | Subscribe → Create AI SDR → Warmup starts |
| **Day 1-2** | Upload 2-3 case studies as artifacts |
| **Day 2-3** | Browse outreach ideas → Build your first offering |
| **Day 3-5** | Launch first campaign using the 5-step wizard |
| **Day 5-14** | Build more offerings, launch more campaigns (warmup pace) |
| **Day 14+** | Full volume! Check Dashboard + Interactions daily |
